Guest Commands

Guest commands are available to visitors and new members so they can explore the clan and the bot safely.

General info

  • /about – Learn what Old School Tool Set is and what it can do for the clan.
  • /help – See a short overview of the main guest commands and how to use them.
  • /invite – Get an invite link so you can add Old School Tool Set to your own server.

OSRS & clan tools

  • !apply – Apply to the clan or submit a join request.
  • /clan – View basic information about the clan, such as name, home world, and focus.
  • /osrs – Get the current time for Old School RuneScape.
  • /rsn – Link an OSRS account name so the bot can recognize you.
  • /wiki – Search the OSRS Wiki for quick answers about items, bosses, and more.
