Owner Commands

These commands are restricted to the bot owner and are not visible to regular users.

Core owner controls

  • !shutdown – Safely shut down the bot instance.
  • !eval <code> – Run owner‑only evaluation snippets.
  • !reload <module> – Reload a command module without restarting.

Security & overrides

  • !blacklist <user> – Block a user from using any commands.
  • !whitelist <user> – Allow a user to bypass certain checks.
  • !sudo <user> <command> – Run a command as another user.
