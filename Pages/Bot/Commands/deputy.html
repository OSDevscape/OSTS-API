Deputy Owner Commands

Deputy Owners can manage configuration and high‑level moderation, but do not have full owner access.

Configuration

  • !config – View current guild configuration.
  • !config set <key> <value> – Update a config value.
  • !prefix <newPrefix> – Change the bot prefix (if you use one).

System tools

  • !stats – View bot stats and health.
  • !logs – Show recent moderation/system events.
