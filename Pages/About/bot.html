About Old School Tool Set

Old School Tool Set is a Discord bot built for Old School RuneScape clans and communities, with a focus on Slayer diaries, clan ranks, and lore‑friendly utilities.

What it’s built for

  • Provide fast access to Slayer Diary entries and monster information.
  • Offer a customizable clan ranking system integrated with the OSTS API.
  • Support events, reminders, and light moderation tools for clan staff.

Data sources

  • Wise Old Man for clan tracking, competitions, and most clan statistics.
  • The OSTS API for Slayer Diaries and clan rank data.
  • Old School RuneScape Wiki and related community resources where applicable.
